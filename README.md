# FLG-Firefox-Extension
Fake Login Generator Chrome Extension allows users to quickly generate temporary login credentials such as email addresses, phone numbers, and secure passwords for testing, development, or privacy purposes.
This tool is designed to help users protect their personal information when interacting with websites that require registration. By generating temporary credentials, users can avoid exposing their real email addresses, phone numbers, or personal data.

Privacy is a core principle of this extension. We do not collect, store, or process personal user data. The extension operates locally in your browser and only communicates with external services when required to retrieve publicly available temporary email or SMS messages.

We strongly believe in minimizing unnecessary data collection and preventing the mass harvesting of personal information. Our goal is to provide a lightweight, transparent, and privacy-respecting tool that helps users maintain control over their digital identity while browsing the web.

No tracking, no analytics, and no background data collection are performed by this extension.

# 🚀 Installation / Instalación

On Linux:
```
git clone https://github.com/Ander-Velc/FLG-Chrome-Extension.git
```
On Windows:

Download the file.zip: https://github.com/Ander-Velc/FLG-Firefox-Extension/archive/refs/heads/main.zip

Extract the file and upload the extension into firefox following these steps:
Enter this address on firefox:
```
about:debugging
```
